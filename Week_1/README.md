# Online Retail II Data Analysis

This project contains exploratory data analysis (EDA) of the Online Retail II dataset.

## Project Structure
- `Week_1/online_retail_II.xlsx`: The raw dataset containing transaction sheets for years 2009-2010 and 2010-2011.
- `Week_1/online_retail_analysis.ipynb`: The Jupyter Notebook containing the data quality audit, visualizations, and business insights.
- `Week_1/documentation.md`: Markdown step-by-step documentation.
- `Week_1/documentation.docx`: Microsoft Word document with embedded screenshots.
- `Week_1/ScreenShots/`: Screenshots of task outputs.

## Setup & Running
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Open and run the Jupyter notebook:
   ```bash
   jupyter notebook Week_1/online_retail_analysis.ipynb
   ```
